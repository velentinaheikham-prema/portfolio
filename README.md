# Portfolio Website

A professional portfolio website showcasing web development skills and projects.

## Technologies

- HTML5
- CSS3
- JavaScript (Vanilla)

## Features

- Responsive design for mobile and desktop
- Smooth navigation with anchor links
- Social media links
- Project showcase
- Contact section
- Professional styling with transitions and hover effects

## Getting Started

1. Open `index.html` in your web browser
2. Or use a local server: `python -m http.server 5500`
3. Navigate to `http://127.0.0.1:5500/`

## Sections

- **About Me** - Introduction and background
- **Technical Skills** - Frontend skills and tools
- **Featured Projects** - Showcase of work
- **Experience** - Work history timeline
- **Contact** - Get in touch options
